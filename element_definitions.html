Element Definitions

h1
Header 1 element, used as the most important title in your hierarchy.
<h1> Header 1 example. </h1>
h2
Header 2 element, used as the 2nd title in your hierarchy.
<h2> Header 2 example. </h2>
h3
Header 3 element, used as the 3rd title in your hierarchy.
<h3> Header 3 example. </h3>
p
Paragraph element, creates a paragraph and holds blocks of text.
<p> This is an example of a paragraph. </p>
ul
An unorganized list element, used to create bulleted lists.
<ul>
<li> Bulleted list item one. </li>
<li> Bulleted list item two. </li>
</ul>
ol
An organized list, used to create numbered lists.
<ol>
<li> Numbered list item one. </li>
<li> Numbered list item two. </li>
</ol>
li
A list element, used inside of the ul or ol elements to add list items.
<ul>
<li> This is a list item. </li>
<li> This is also a list item. </li>
</ul>
