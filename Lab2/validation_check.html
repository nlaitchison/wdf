Validation Check - Lab 2

FearlessRecords.com

Fearless Records is a record company and their site consists of news updates, band information, and videos.

The markup has 20 errors in the validator.nu and the HTML Lint Validator couldn’t continue because of an error in the HTML.

A lot of errors occurred because entities weren’t used for the ampersand. This can be easily fixed by replacing the ampersands with it's entity. The other errors occurred because of missing alt attribute values. All the images have alt=” “, but aren’t given any values. The HTML Lint Validator couldn’t be continued because inline elements, <strong> <p> <br />, were used in the values for title attributes.

PureNoise.net

Pure Noise Records is also a record company and their site consists of news updates, band information, and videos.

The markup has 25 errors in the validator.nu and the HTML Lint Validator couldn’t continue because of an error in the HTML.

End tags aren’t in order throughout the page. <p><font color="#666666">No More Than This</p></font> So the end tags just need to be flipped around to fix that. This is also the reason why the HTML Lint Validator couldn’t continue the validation. All the images need alt attributes so that screen readers can read the entire page’s content. Also, for some elements, spaces are missing in between separate attributes.
